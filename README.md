# 🏥 Sistema Hospitalar VidaPlus (SGHSS)

Sistema web desenvolvido para gerenciamento hospitalar, com foco em organização de atendimentos, pacientes e serviços de saúde.

---

## 🚀 Funcionalidades

* 🔐 Login de usuários
* 📊 Dashboard administrativo
* 🧑‍⚕️ Cadastro de pacientes
* 📅 Agenda de consultas
* 💻 Telemedicina

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
